Gendiff.
Work with JSON and YML.

### Install
```
npm i
```
### Usage
Help:
```
gendiff -h
```
Version:
```
gendiff -V
```
Compare:
```
gendiff <path to file1> <path to file2>
```
Select the output format:
```
gendiff --format plain <path to file1> <path to file2>
```
